Introduction

SPA - 单页面应用程序

1.路由

  • 路由指的是URL地址与程序的映射关系
      http://boxuegu.com/login
  http://boxuegu.com/tracher/manage
  这是后端路由,地址与后端程序的一种映射
  • AngularJS的路由,仍然是URL地址与程序的映射关系,只不过前端路由,是URL地址与前端程序映射关系.
  • 一个应用是由若干个视图组成,根据不同的业务逻辑展示给用户不同的视图,路由则是实现这一功能的关键.

2. SPA - Single Page Application

  • SPA-单页面应用程序指的是整个网站程序只有一个页面,通过局部刷新(Ajax异步请求)来实现功能的切换.
  • 单页面网站案例
    • 网易云音乐
    • 小剧客栈
  • 单页应用程序的特点
    • 你会很明显发现,这样的网站用户体验极好.它们的界面切换非常流畅，响应很迅速，跟传统的网页明显不一样.
    • 所谓单页应用，指的是在一个页面上集成多种功能，甚至整个系统就只有一个页面，所有的业务功能都是它的子模块，通过特定的方式挂接到主界面上。它是AJAX技术的进一步升华，把AJAX的无刷新机制发挥到极致，因此能造就与桌面程序媲美的流畅用户体验。

      3. 猜想这样的网站功能是如何实现的

  • 点击的其实是一个锚链接.
  • 在锚链接被点击的时候,浏览器不会主动刷新页面.
  • 而是我们在锚链接点击的时候,监听点击事件.
  • 动态的异步请求数据 局部刷新页面.

    4. 尝试自己实现

  • 实现思路
    • 监听页面锚点值的变化. 单击锚链接
    • 根据不同的锚点来请求数据以Ajax的方式局部展示在页面上.
  • 仿网易云音乐
    • 监听锚点值变化的事件 - hashchange
    • 通过location对象的hash属性的值可以拿到当前锚点.
      window.onhashchange = function () {
  var hash = window.location.hash.slice(1);
  var url = '';
  switch(hash){
      case 'home':
          url = 'data/home.json';
          break;
      case 'my':
          url = 'data/my.json';
          break;
      case 'find':
          url = 'data/find.json';
          break;
  }
  var xhr = new XMLHttpRequest();
  xhr.open('get',url);
  xhr.onreadystatechange = function () {
      if(xhr.readyState==4 && xhr.status==200){
          var obj = JSON.parse(xhr.responseText)
          document.getElementById('content').innerText = obj.content;
      }
  }
  xhr.send();
}
